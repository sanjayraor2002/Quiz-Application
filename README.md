# Quiz Application

A complete Quiz Application built using **Java**, **Hibernate**, and **MySQL**. This application allows users to create, manage, and participate in quizzes with multiple-choice questions. It uses Hibernate for database interactions and follows a straightforward object-oriented design.

## Features
- **Create Questions**: Add quiz questions along with multiple-choice options.
- **Set Correct Answers**: Assign the correct answer to each question.
- **Manage Options**: Create and manage multiple options for each question.
- **Database Storage**: Store all quiz data (questions, answers, and options) in a MySQL database.
- **User-Friendly**: Simple command-line interface for easy interaction.

## Technologies Used
- **Java**: Core programming language.
- **Hibernate**: ORM tool for managing database operations.
- **MySQL**: Relational database for storing quiz data.
- **Maven**: Dependency management and build automation tool.

## Getting Started

### Prerequisites
- **Java Development Kit (JDK)** installed on your machine.
- **MySQL Server** for the database.
- **Maven** for building the project.
